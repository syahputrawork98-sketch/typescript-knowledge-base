# SR-06: Utility Types

> *"Utility Types adalah perpustakaan standar TypeScript di level tipe — memahami implementasinya mengajarkan semua yang perlu diketahui tentang type-level programming."*

## Gambaran Sub-Rak
Utility Types adalah koleksi tipe generik bawaan TypeScript. Sub-Rak ini tidak hanya membahas cara pakainya, tetapi juga implementasi manual di balik setiap utility — karena dari situ kamu belajar cara membuat utility types sendiri.

**Source:** [TypeScript Handbook — Utility Types](https://www.typescriptlang.org/docs/handbook/utility-types.html)

## Struktur Buku (BK)

| Buku | Slug | Konten Bab-Bab di Dalamnya |
|:---|:---|:---|
| **BK-01** | `TransformationUtils` | `Partial`, `Required`, `Readonly`, `Record`, `Pick`, `Omit` — implementasi manual dengan Mapped Types |
| **BK-02** | `ExtractionUtils` | `Exclude`, `Extract`, `NonNullable`, `ReturnType`, `Parameters`, `InstanceType` — implementasi dengan Conditional Types |
| **BK-03** | `CompositionPatterns` | Komposisi utility types, membuat custom utility, deep partial patterns, branded types |
