# SR-02: Declaration Files

> *"`.d.ts` adalah bayangan tipe dari JavaScript — ia adalah cara TypeScript berbicara dengan dunia yang tidak bertipe."*

## Gambaran Sub-Rak
Declaration files (`.d.ts`) adalah cara TypeScript merepresentasikan tipe untuk kode yang tidak memiliki sumber TypeScript — baik library JavaScript lama, modul native, maupun API global browser/Node.js.

**Source:** [TypeScript Handbook — Declaration Files](https://www.typescriptlang.org/docs/handbook/declaration-files/introduction.html)
