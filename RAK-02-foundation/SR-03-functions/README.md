# SR-03: Functions

> *"Fungsi TypeScript bukan sekadar fungsi JavaScript + tipe — ia membawa overloads, call signatures, dan this-parameter yang mengubah cara kamu berpikir tentang kontrak kode."*

## Gambaran Sub-Rak
Fungsi adalah warga kelas satu di TypeScript. Sub-Rak ini membahas cara TypeScript merepresentasikan, memeriksa, dan menyimpulkan tipe fungsi — termasuk kasus-kasus edge yang sering membingungkan bahkan developer berpengalaman.

**Source:** [TypeScript Handbook — More on Functions](https://www.typescriptlang.org/docs/handbook/2/functions.html)
