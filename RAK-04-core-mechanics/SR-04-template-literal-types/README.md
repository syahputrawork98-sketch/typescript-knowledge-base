# SR-04: Template Literal Types

> *"Template Literal Types mengubah TypeScript menjadi bahasa yang bisa memprogram string di level tipe — sebuah kemampuan yang tidak ada di bahasa statically-typed lain."*

## Gambaran Sub-Rak
Template Literal Types memungkinkan konstruksi dan manipulasi string secara deklaratif di level tipe — bukan di runtime. Ini membuka kemampuan seperti membuat union tipe dari kombinasi string secara otomatis.

**Source:** [TypeScript Handbook — Template Literal Types](https://www.typescriptlang.org/docs/handbook/2/template-literal-types.html)

## Struktur Buku (BK)

| Buku | Slug | Konten Bab-Bab di Dalamnya |

| **BK-01** | `TemplateLiteralBasics` | `` `${Type}` `` syntax, interpolasi union types, contoh penggunaan `EventName` patterns |
| **BK-02** | `StringManipulationTypes` | `Uppercase<S>`, `Lowercase<S>`, `Capitalize<S>`, `Uncapitalize<S>`, komposisi string types |
