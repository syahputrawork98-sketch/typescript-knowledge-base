# SR-06: Enums

> *"Enum TypeScript adalah satu-satunya fitur yang menghasilkan kode runtime — bukan sekadar type annotation. Ini membuat enum unik sekaligus penuh dengan gotcha."*

## Gambaran Sub-Rak
Enum adalah fitur paling kontroversial di TypeScript. Tidak seperti fitur lain yang hanya ada saat compile-time, enum menghasilkan JavaScript nyata. Sub-Rak ini membahas semua variannya, pitfalls-nya, dan kapan lebih baik menggunakan alternatif.

**Source:** [TypeScript Handbook — Enums](https://www.typescriptlang.org/docs/handbook/enums.html)
