# SR-05: Type Inference

> *"Inferensi adalah kecerdasan compiler — TypeScript sering tahu tipe yang lebih benar dari yang kamu pikirkan."*

## Gambaran Sub-Rak
Type Inference adalah kemampuan `tsc` untuk menentukan tipe secara otomatis tanpa anotasi eksplisit. Memahami algoritmanya mencegah kamu menambah anotasi yang tidak perlu dan menghindari tipe yang terlalu luas.

**Source:** [TypeScript Handbook — Type Inference](https://www.typescriptlang.org/docs/handbook/type-inference.html)
