# SR-07: Modules

> *"TypeScript modules bukan hanya ES import/export — ada lapisan resolusi, ambient declarations, dan augmentation yang membentuk fondasi ekosistem @types."*

## Gambaran Sub-Rak
Module system TypeScript mencakup cara `tsc` menemukan, memuat, dan memverifikasi tipe dari modul — baik lokal maupun dari `node_modules`. Memahami ini adalah prasyarat untuk RAK-05 (Ecosystem).

**Source:** [TypeScript Handbook — Modules](https://www.typescriptlang.org/docs/handbook/2/modules.html)
