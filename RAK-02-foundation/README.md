# RAK-02: Foundation — The Standard Book

> **"Handbook adalah kontrak. Baca dengan presisi arsitek, bukan sekadar pengguna."**

Rak kedua adalah referensi sintaks sehari-hari TypeScript. Berdasarkan langsung pada [TypeScript Handbook](https://www.typescriptlang.org/docs/handbook/), setiap unit di sini membahas fitur secara mendalam: *apa itu*, *bagaimana cara kerjanya di level tipe*, dan *kapan menggunakannya dengan benar*.

## Struktur Sub-Rak

| Sub-Rak | Slug | Konten |
|:---|:---|:---|
| **SR-01** | `primitives-and-basics` | `string`, `number`, `boolean`, `null`, `undefined`, `symbol`, `bigint`, `any`, `unknown`, `never`, `void` |
| **SR-02** | `narrowing` | `typeof`, `instanceof`, `in`, *Equality Narrowing*, *Control Flow Analysis*, *Type Predicates*, *Discriminated Unions* |
| **SR-03** | `functions` | Function Types, Optional & Default Params, Rest Params, Overloads, `this` parameter, *Call Signatures* |
| **SR-04** | `objects-and-interfaces` | Object Types, Interface vs Type Alias, *Index Signatures*, *Readonly*, `extends`, *Intersection Types* |
| **SR-05** | `classes` | Class Fields, Constructor, Access Modifiers, `abstract`, `implements`, *Parameter Properties* |
| **SR-06** | `enums` | `const enum` vs regular `enum`, *Heterogeneous Enum*, *Enum as Type*, pitfalls enum |
| **SR-07** | `modules` | ES Modules di TS, `import type`, *Ambient Modules*, `/// <reference>` directives |
