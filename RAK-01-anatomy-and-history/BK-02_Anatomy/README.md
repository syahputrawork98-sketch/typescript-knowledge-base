# BK-02: Anatomy — The Structure of TypeScript

> *"Mengenal anatomi sebuah bahasa adalah seperti membaca blueprint gedung sebelum masuk ke dalamnya."*

**Unit ini tidak membutuhkan Lab Praktis/Visualisasi karena bersifat penjelasan sejarah/konsep naratif.**

## Gambaran Buku Ini
Buku kedua membahas **anatomi TypeScript sebagai bahasa**: bagaimana ia disusun, komponen-komponen apa yang membentuknya, dan bagaimana hubungannya dengan ekosistem JavaScript yang lebih besar.

## Bab-Bab dalam Buku Ini

| Bab | Slug | Topik |
|:---|:---|:---|
| **CH-01** | `TSEcosystemMap` | Peta ekosistem TypeScript: `tsc`, `tsconfig`, `.d.ts`, `@types`, dan hubungan antar komponen |
| **CH-02** | `SupersetLayer` | Bagaimana TypeScript "membungkus" JavaScript — layer superset dan batas kemampuannya |
| **CH-03** | `TypeAnnotationAnatomy` | Anatomi anotasi tipe: `variable: Type`, parameter, return types, dan type assertions |
| **CH-04** | `CompilationFlow` | Alur high-level dari `.ts` → type-check → `.js`: apa yang terjadi saat kamu tekan compile |
