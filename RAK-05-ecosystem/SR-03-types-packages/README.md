# SR-03: @types Packages

> *"DefinitelyTyped adalah proyek komunitas terbesar dalam sejarah TypeScript — ribuan `.d.ts` untuk ribuan library JavaScript."*

## Gambaran Sub-Rak
`@types/*` adalah cara standar mendapatkan type definitions untuk library JavaScript. Sub-Rak ini membahas cara membaca, menggunakan, dan bahkan menulis sendiri type packages.

**Source:** [DefinitelyTyped](https://github.com/DefinitelyTyped/DefinitelyTyped) | [TypeScript Docs — @types](https://www.typescriptlang.org/docs/handbook/2/type-declarations.html)

## Struktur Buku (BK)

| Buku | Slug | Konten Bab-Bab di Dalamnya |
|:---|:---|:---|
| **BK-01** | `UsingTypes` | Instalasi `@types/*`, cara `tsc` menemukan types, `typeRoots`, `types` option di tsconfig |
| **BK-02** | `WritingTypes` | Membaca `.d.ts` yang kompleks, Module Augmentation untuk extend library, menulis custom `@types` |
