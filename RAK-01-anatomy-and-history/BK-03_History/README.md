# BK-03: History — The Origin Story

> *"Memahami sejarah sebuah bahasa mencegah kita mengulangi kesalahan masa lalu dan memberikan konteks untuk keputusan desain masa kini."*

**Unit ini tidak membutuhkan Lab Praktis/Visualisasi karena bersifat penjelasan sejarah/konsep naratif.**

## Gambaran Buku Ini
Buku ketiga menelusuri **sejarah lahirnya TypeScript** — dari frustrasi membangun aplikasi web berskala besar dengan JavaScript, hingga pengumuman open-source di PDC 2012, dan adopsi global yang mengubah industri.
