# TypeScript Knowledge Base: Universal Deconstruction Plan

> **Status**: Gold Standard (6-Level Hierarchy)
> **Last Updated**: 2026-03-22

Rencana ini menggunakan **Unified Gold Standard** untuk mentransformasi dokumen teknis menjadi unit pelajaran yang presisi melalui hirarki **6-Level Depth**.

---

## 1. Arsitektur 6-Rak (The Universe Blueprint)

### I. RAK-01-anatomy (The Landscape)
Narasi awal penciptaan (superset dari JS), pro dan kontra keharusan tipisasi, serta miskonsepsi umum.

### II. RAK-02-foundation (The Standard Book)
Sintaks dan anomali tipisasi TypeScript (Primitives, Narrowing, Objects, Arrays, Tuples).

### III. RAK-03-evolution (History & Future)
Analisis mendetail rilisan `tsc` (TypeScript Compiler) dan modernisasi fitur terbarunya.

### IV. RAK-04-core-mechanics (The Internal Logic)
Buku besar sistem tipe struktural: *Generics*, *Mapped Types*, *Conditional Types*, *Type Inference*, dan algoritma resolusi tipe.

### V. RAK-05-ecosystem (The Environment)
Integrasi Lingkungan TS, deklarasi `@types` (`.d.ts`), tsconfig konfigurasional, serta perannya dengan *Bundlers* modern.

### VI. RAK-06-compiler (The Machine Room)
Deep dive ke ruang mesin **`tsc`** (TS Compiler API): Lexing, AST Generation, *Type Checking Stages*, dan emisi (Transformasi Transpiler).
