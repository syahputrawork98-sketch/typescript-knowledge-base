# SR-02: Domain-Driven Design

> *"DDD memberi kita bahasa untuk berbicara dengan domain expert. TypeScript memberi kita alat untuk mengkodekan bahasa itu dengan presisi tipe."*

## Gambaran Sub-Rak
Domain-Driven Design (DDD) dalam TypeScript: memodelkan proses bisnis nyata menggunakan Value Objects, Entities, Aggregates, dan Domain Events — semuanya dijamin konsistensinya oleh type system.

## Struktur Buku (BK)

| Buku | Slug | Konten Bab-Bab di Dalamnya |
|:---|:---|:---|
| **BK-01** | `DDDBuildingBlocks` | Value Objects (immutable, equality by value), Entities (identity), Aggregates (consistency boundary) |
| **BK-02** | `DDDInTypeScript` | Branded types untuk Value Objects, readonly untuk immutability, Domain Events typing, Repository interfaces |
