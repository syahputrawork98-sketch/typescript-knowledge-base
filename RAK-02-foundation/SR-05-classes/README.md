# SR-05: Classes

> *"Class TypeScript bukan hanya sintaks ES6+ — ia membawa layer type-checking yang mengubah class menjadi blueprint yang diverifikasi compiler."*

## Gambaran Sub-Rak
TypeScript memberi class JavaScript kemampuan baru: access modifiers, abstract classes, parameter properties, dan `implements` untuk membuktikan kontrak interface. Sub-Rak ini membedah semuanya sampai level emitted JavaScript.

**Source:** [TypeScript Handbook — Classes](https://www.typescriptlang.org/docs/handbook/2/classes.html)
