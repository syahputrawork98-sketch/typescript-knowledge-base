# SR-01: Generics

> *"Generic bukan sintaks — ia adalah fungsi yang beroperasi di ranah tipe. `identity<T>(x: T): T` adalah program tipe paling sederhana yang pernah ada."*

## Gambaran Sub-Rak
Generics adalah fondasi dari reusability di TypeScript. Sub-Rak ini membahas bagaimana generics bekerja pada level algoritma type inference, bukan hanya cara penggunaannya.

**Source:** [TypeScript Handbook — Generics](https://www.typescriptlang.org/docs/handbook/2/generics.html)
