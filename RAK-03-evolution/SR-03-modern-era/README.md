# SR-03: Modern Era — TypeScript 5.x

> *"TypeScript 5.x adalah kedewasaan: Decorator resmi, const type parameters, dan peningkatan performa compiler yang masif."*

**Unit ini tidak membutuhkan Lab Praktis/Visualisasi karena bersifat penjelasan sejarah/konsep naratif.**

## Gambaran Sub-Rak
Era modern TypeScript: rilis 5.x membawa stabilisasi fitur-fitur eksperimental dan penambahan kemampuan baru yang long-awaited — termasuk Decorator Stage 3, `const` type parameters, dan keyword `using`.

## Struktur Buku (BK)

| Buku | Slug | Konten Bab-Bab di Dalamnya |
|:---|:---|:---|
| **BK-01** | `TS5x-CoreFeatures` | TS 5.x: `const` Type Parameters, *Decorator* (TC39 Stage 3), `satisfies` operator, `using` & `await using` |
| **BK-02** | `TS5x-Improvements` | Performance improvements, `moduleResolution: bundler`, `verbatimModuleSyntax`, ESM-native support |
