# SR-04: Objects & Interfaces

> *"Interface adalah kontrak. Type Alias adalah alias. Perbedaannya bukan sekadar sintaks — ini tentang kapabilitas dan declaration merging."*

## Gambaran Sub-Rak
Objects dan Interfaces adalah cara utama TypeScript memodelkan struktur data. Sub-Rak ini membahas perbedaan mendalam antara `interface` dan `type`, Index Signatures, Structural Typing, dan komposisi tipe melalui intersection.

**Source:** [TypeScript Handbook — Object Types](https://www.typescriptlang.org/docs/handbook/2/objects.html)

| Buku | Slug | Konten Bab-Bab di Dalamnya |
